##  Deskripsi Singkat
Proyek ini adalah **generator biodata pendaftaran mahasiswa** berbasis web.  
User dapat mengisi formulir (nama, TTL, jenis kelamin, prodi, email, dan telepon),  
dan hasilnya langsung ditampilkan secara **real-time** di area **preview** di bawah formulir.

---

##  Teknologi yang Digunakan
- HTML5  
- CSS3 (Flexbox + Responsif)  
- JavaScript (DOM Manipulation & Event Listener)

---

##  Tampilan
### Mobile (1 Kolom)
Form dan preview ditampilkan vertikal.

### Desktop (2 Kolom)
Formulir di kiri, hasil biodata di kanan menggunakan Flexbox.

---

##  Cara Menjalankan
1. Buka file `index.html` di browser.  
2. Isi formulir dengan data pribadi.  
3. Lihat hasil biodata langsung muncul di area preview.
